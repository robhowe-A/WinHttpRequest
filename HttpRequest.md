# HttpRequest  
This repository stores a windows application project (Desktop + .NET + C#). It provides a gui to send http requests and displays the response data.  

Developed in forms style, this app is responsive on windows desktops, expands/collapses menus on request or need, and dynamically adapts controls based on different situations the program offers.  

## Testers  
- An executable of the application is made up of this repository's code. Just click on the release tag from the code page to download the application. It's target is for windows x64 applications.
